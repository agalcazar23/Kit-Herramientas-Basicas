**LocalSend** es una solución de código abierto diseñada para compartir archivos y mensajes de forma segura entre dispositivos cercanos a través de la red local, sin necesidad de internet ni de servidores externos.

[🌐 Sitio Web Oficial](https://localsend.org) • [📂 Código Fuente](https://github.com/localsend/localsend) • [📖 Documentación](https://localsend.org/#/docs)

### 💎 ¿Por qué LocalSend supera a cualquier alternativa?

> [!IMPORTANT]
> **Privacidad Total:** A diferencia de WeTransfer o AirDrop, tus datos nunca salen de tu red local. Es comunicación pura P2P (Peer-to-Peer) con cifrado HTTPS.

| ⚡ AGILIDAD | 🛡️ SEGURIDAD | 🌍 UNIVERSAL |
| :--- | :--- | :--- |
| Sin necesidad de registros. Instalar y enviar. | Cifrado TLS/SSL integrado en cada transferencia. | Funciona entre un PC Windows, un móvil Android y un Mac simultáneamente. |

---

### 🛠️ Capacidades Técnicas Resumidas

| Función | Descripción Técnica |
| :--- | :--- |
| **Zero Config** | Utiliza el protocolo de descubrimiento mDNS para encontrar dispositivos automáticamente. |
| **Multi-Envío** | Permite enviar el mismo paquete de datos a múltiples objetivos de forma paralela. |
| **Cifrado** | Implementa certificados TLS dinámicos para proteger la integridad de los archivos. |
| **Modo Rápido** | Aceptación automática de archivos para flujos de trabajo de alta intensidad. |

---

### 🚀 Guía de Despliegue Rápido

1. **Sincronización:** Asegúrate de que los dispositivos estén en la misma red Wi-Fi o Ethernet.
2. **Selección:** Arrastra los archivos a la interfaz de LocalSend.
3. **Envío:** Selecciona el alias del dispositivo destino (Ej: `Fancy Apple` o `Great Orange`).
4. **Finalización:** ¡Listo! Ahora puedes transferir datos sin cables, sin nubes y sin límites de tamaño.
